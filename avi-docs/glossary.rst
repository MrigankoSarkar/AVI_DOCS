Glossary
========

API Key
-------
A unique credential pair used to authorize requests to the API surface.

ETL
---
Extraction, Transformation, and Loading process used to ingest and normalize data into the database.

MDDS
----
Metadata Directory of Standardized Data, the governance model for Indian administrative datasets.

Neon
----
A serverless Postgres provider used by the project.

Redis
-----
An in-memory data store used for caching, job coordination, and rate limiting.
