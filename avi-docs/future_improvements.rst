Future Improvements
===================

The project is production-ready but can be strengthened further with the following improvements:

- Add dedicated unit and integration tests for both the Node application and Python ETL service.
- Implement authentication on the Python ETL upload endpoints.
- Add OpenAPI/Swagger documentation for the API surface.
- Add a production Dockerfile for the Node app and the Python service.
- Add explicit migration tooling for Drizzle and schema versioning.
- Add observability dashboards for Redis, Postgres, and request latency.
- Add a contributor guide and release process documentation.
